# Projeto de Realidade Aumentada com Unity e AR Foundation

## 📖 Descrição

Este projeto foi desenvolvido utilizando o **Unity 2022 LTS** e o **AR Foundation** com o objetivo de criar uma aplicação de Realidade Aumentada (AR) para dispositivos Android.

A aplicação detecta superfícies planas no ambiente através da câmera do celular e permite ao usuário posicionar um objeto 3D tocando na tela.

---

## 🎯 Objetivos

- Desenvolver uma aplicação simples de Realidade Aumentada.
- Utilizar o AR Foundation para detecção de planos.
- Posicionar objetos 3D em superfícies reais.
- Gerar um APK funcional para dispositivos Android.

---

## 🛠 Tecnologias Utilizadas

- Unity 2022 LTS
- C#
- AR Foundation
- ARCore XR Plugin
- Android Build Support

---

## 📱 Funcionalidades

- Abertura da câmera do dispositivo.
- Detecção automática de superfícies planas.
- Posicionamento de objeto 3D por toque na tela.
- Geração de APK para Android.

## ▶️ Como Executar

### Requisitos

- Unity 2022 LTS
- Android Build Support
- Dispositivo Android compatível com ARCore

### Passos

1. Clone este repositório.
2. Abra o projeto no Unity 2022 LTS.
3. Selecione a plataforma Android em **File > Build Settings**.
4. Conecte um dispositivo Android com Depuração USB habilitada.
5. Clique em **Build and Run**.
